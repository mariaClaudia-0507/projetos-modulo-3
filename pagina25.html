Frontend
Front end é a parte visual e interativa de um site, aplicativo ou software com a qual o usuário interage diretamente. Desenvolvido com HTML (estrutura), CSS (estilo) e JavaScript (interatividade), o front end transforma o layout criado por designers em uma interface funcional, focada na experiência do usuário (UX).
